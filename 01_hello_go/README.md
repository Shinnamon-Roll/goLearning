# บทที่ 01: Hello Go! 👋

## 📖 บรรยาย (Lecture)

### Go คืออะไร?

Go (หรือ Golang) เป็นภาษาโปรแกรมที่สร้างโดย Google มีจุดเด่นคือ:
- **คอมไพล์แล้ว (Compiled)** — ได้ไฟล์ binary รันเร็ว ไม่ต้องลง runtime เพิ่ม
- **Garbage Collector** — ไม่ต้องจัดการหน่วยความจำเอง
- **Concurrency ในภาษา** — goroutine + channel ใช้งานง่าย
- **ไวยากรณ์เรียบง่าย** — keyword น้อย เรียนรู้เร็ว

### โปรแกรม Go แรกของคุณ

ทุกโปรแกรม Go เริ่มจาก `package main` และฟังก์ชัน `main`:

```go
package main

import "fmt"

func main() {
    fmt.Println("สวัสดี Go!")
}
```

**อธิบายทีละบรรทัด:**
- `package main` — บอก Go ว่านี่คือโปรแกรมหลัก (ไม่ใช่ library)
- `import "fmt"` — ดึงแพ็กเกจ `fmt` มาใช้ (format I/O)
- `func main()` — จุดเริ่มต้นของโปรแกรม Go จะรันฟังก์ชันนี้ก่อนเสมอ
- `fmt.Println(...)` — พิมพ์ข้อความพร้อมขึ้นบรรทัดใหม่

### คำสำคัญที่ต้องรู้

| คำ | ความหมาย |
|-----|-----------|
| `package main` | จุดเริ่มต้นของโปรแกรม (entry point) |
| `import` | ดึงแพ็กเกจอื่นมาใช้ |
| `func main()` | ฟังก์ชันที่รันเป็นอันดับแรก |
| `fmt` | แพ็กเกจสำหรับจัดรูปแบบ input/output |

### ฟังก์ชันสำคัญใน `fmt`

| ฟังก์ชัน | หน้าที่ | ตัวอย่าง | ผลลัพธ์ |
|----------|--------|---------|---------|
| `fmt.Println(a, b)` | พิมพ์พร้อมขึ้นบรรทัดใหม่ | `Println("สวัสดี", "โลก")` | `สวัสดี โลก` |
| `fmt.Printf(format, args)` | พิมพ์แบบระบุฟอร์แมต | `Printf("อายุ %d", 25)` | `อายุ 25` |
| `fmt.Sprintf(format, args)` | สร้าง string จากฟอร์แมต (ไม่พิมพ์) | `Sprintf("อายุ %d", 25)` | `"อายุ 25"` |
| `fmt.Scanln(&var)` | อ่านค่าจาก keyboard | `Scanln(&name)` | — |

### Format Verbs (ตัวระบุรูปแบบ)

| Verb | ใช้กับ | ตัวอย่าง | ผลลัพธ์ |
|------|--------|----------|---------|
| `%v` | ทุกชนิด (ค่าปริยาย) | `Sprintf("%v", 42)` | `"42"` |
| `%s` | string | `Sprintf("%s", "Go")` | `"Go"` |
| `%d` | integer (เลขจำนวนเต็ม) | `Sprintf("%d", 14)` | `"14"` |
| `%f` | float (เลขทศนิยม) | `Sprintf("%f", 3.14)` | `"3.140000"` |
| `%t` | bool (จริง/เท็จ) | `Sprintf("%t", true)` | `"true"` |
| `%T` | แสดง **ชนิด** ข้อมูล | `Sprintf("%T", 42)` | `"int"` |

## 🧪 กรณีศึกษา (Study Cases)

### กรณีศึกษา 1: ฟังก์ชันทักทาย — `greet`

**โจทย์:** ร้านกาแแฟต้องการระบบทักทายลูกค้า สร้างฟังก์ชันที่รับชื่อแล้วคืนข้อความทักทาย

```go
// ตัวอย่าง:
greet("สมชาย") → "Hello, Go! Welcome, สมชาย!"
greet("Go")     → "Hello, Go! Welcome, Go!"
```

**แนวคิด:**
- ประกาศฟังก์ชัน: `func ชื่อ(พารามิเตอร์) ชนิดคืนค่า { ... }`
- ต่อ string ด้วย `+`: `"สวัสดี" + " " + name`
- หรือใช้ `fmt.Sprintf`: `fmt.Sprintf("Hello, Go! Welcome, %s!", name)`

### กรณีศึกษา 2: จัดรูปแบบข้อความ — `printFormatted`

**โจทยะ:** ระบบลงทะเบียนต้องการแสดงข้อมูลผู้ใช้ สร้างฟังก์ชันที่รับชื่อและอายุ คืน string ที่จัดรูปแบบแล้ว

```go
// ตัวอย่าง:
printFormatted("สมชาย", 25) → "สมชาย is 25 years old"
printFormatted("Go", 14)      → "Go is 14 years old"
```

**แนวคิด:**
- ใช้ `fmt.Sprintf` กับ format verbs
- `%s` = string, `%d` = integer
- `fmt.Sprintf("%s is %d years old", name, age)`

### กรณีศึกษา 3: สำรวจชนิดข้อมูล — `summarizeTypes`

**โจทย์:** สร้างฟังก์ชันที่แสดงความสามารถของ format verbs ทุกชนิด (`%v`, `%s`, `%d`, `%t`, `%T`) อย่างน้อยอย่างละ 1 ครั้ง

```go
// ตัวอย่างผลลัพธ์ (เป็นได้หลายแบบ):
summarizeTypes() → "ชื่อ=Go เวอร์ชัน=14 ใช้งานได้=true ชนิด=int"
// ข้อสอบเช็คแค่ว่า:
// - ผลลัพธ์มีความยาว >= 20 ตัวอักษร
// - มีทั้งตัวอักษรและตัวเลขปนกัน
```

**แนวคิด:**
- ลองประกาศตัวแปรหลายชนิด: `name := "Go"`, `version := 14`, `active := true`
- ใช้ `%v` กับทุกชนิด, `%s` กับ string, `%d` กับ int, `%t` กับ bool, `%T` แสดงชนิด

## 💡 เคล็ดลับ

- Go ไม่มี string interpolation แบบ Python f-string → ต้องใช้ `fmt.Sprintf`
- `%v` ใช้ได้กับทุกชนิดข้อมูล (เหมือน `toString()` ในภาษาอื่น)
- `%T` พิมพ์ **ชนิดข้อมูล** ไม่ใช่ค่า เช่น `fmt.Sprintf("%T", 42)` → `"int"`
- ต่อ string ด้วย `+`: `"Hello" + " " + "Go"` → `"Hello Go"`
- ประกาศตัวแปรแบบสั้น: `x := 42` (เหมือน `var x = 42` แต่สั้นกว่า)
